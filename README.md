# 🩺 AEGIS AI – Intelligent Medical Chatbot & Healthcare Assistant

AEGIS AI is a comprehensive healthcare web application built using **Python**, **Streamlit**, and **SQLite** that provides AI-assisted symptom analysis, medical information, hospital discovery, medicine reminders, and personalized healthcare support.

The application is designed to improve accessibility to healthcare information while demonstrating backend engineering, REST-ready architecture, database management, and responsive web application development.

---

## 🚀 Features

### 👤 User Authentication
- Secure user registration and login
- SHA-256 password hashing
- Patient, Medical Student, and Healthcare Professional roles
- Personalized dashboard

### 🩺 AI Symptom Analyzer
- Rule-based symptom analysis
- Emergency detection
- Severity classification
- Personalized recommendations
- Consultation history

### 💬 Medical Chatbot
- AI-powered medical Q&A
- Context-aware responses
- Medical educational mode
- Healthcare professional mode

### 🏥 Hospital Finder
- Nearby hospital recommendations
- Interactive map using Folium
- Emergency hospital filtering
- Hospital ratings and specialties

### 💊 Medicine Reminder System
- Medicine scheduling
- Water intake reminders
- Notification system
- Medication tracking

### 📄 Medical Report Generator
- PDF consultation reports
- JSON export
- Consultation history
- Health records management

### 📊 Personal Dashboard
- Consultation analytics
- Health statistics
- Activity tracking
- User-specific recommendations

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AEGIS-AI.git
```

Move into the project directory

```bash
cd AEGIS-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run medical_app.py
```
